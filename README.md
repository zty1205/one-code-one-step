## 生成 code 文件和 md 链接

<br/>

运行

```sh
npm run file
```

参数

- open: 是否自动打开 code 文件
- name: 文件名
- ext: 后缀名
- dir: 文件夹

```sh
npm run file --open=false --name=test --ext=html --dir=leetCode
```

文件名支持

- 题目名，多个使用","隔开：生成普通的代码文件
- LeetCode http 链接：将该网页上的信息写入代码文件
